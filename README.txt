POI Lookup
This algorithm takes a XY Point and generates a map of features around the point. 

Input LAT, LON Coordinate: LAT and LON Coordinate of the point. The LAT and LON must be deliminated by either a space or a comma (Required)

Input Map Scale: Scale factor to zoom into the entered point. 50 - 100 is usually a good starting place. Larger number zoom out. The lowest scale is 10.

Input Append LAT LON: This will append the LAN and LON data points to the  output filename.

Input Append Map Scale: This will append the scale factor to the end of the output filename.

Output PDF File: Path and base file name for the PDF output.
